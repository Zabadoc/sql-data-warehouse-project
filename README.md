# Data Warehouse and Analytics

Welkom bij de **Datawarehouse- en analytics-project** opslagplaats!  
Dit project laat een uitgebreide datawarehouse- en analytics-oplossing zien, van het bouwen van een datawarehouse tot het genereren van bruikbare inzichten. Ontworpen als een portfolio-project, benadrukt het de beste praktijken uit de industrie op het gebied van data-engineering en analyse.

---

## Project Vereisten

### Het bouwen van het datawarehouse (Data Engineering)

#### Het doel
Ontwikkel een modern datawarehouse met SQL Server om verkoopgegevens te consolideren, zodat je analytische rapportages kunt maken en betere beslissingen kunt nemen.

#### Specificaties
- **Data Sources**: Gegevens importeren uit twee bronsystemen (ERP en CRM) die als CSV-bestanden zijn aangeleverd.
- **Data Quality**: Maak de data schoon en los kwaliteitsproblemen op voordat je gaat analyseren.
- **Integration**: Combineer beide bronnen tot één gebruiksvriendelijk datamodel dat is ontworpen voor analytische queries.
- **Scope**: Concentreer je alleen op de nieuwste dataset; historisering van gegevens is niet nodig.
- **Documentation**: Zorg voor duidelijke documentatie van het datamodel om zowel zakelijke belanghebbenden als analyseteams te ondersteunen.

---

### BI: Analyse & rapportage (Data-analyse)

#### Het doel
Ontwikkel op SQL gebaseerde analyses om gedetailleerde inzichten te leveren in:
- **Klanten gedrag**
- **Productprestaties**
- **Verkooptrends**

Deze inzichten geven belanghebbenden belangrijke bedrijfsgegevens, waardoor ze strategische beslissingen kunnen nemen.
